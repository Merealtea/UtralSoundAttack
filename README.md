# UltraSoundAttack

## 这个是IOT超声波攻击大作业的整合支撑材料
小组成员：徐翰文、陈星元、陈修元、高弈杰、杨刘佳

code文件夹里包含了攻击、防御、跳频攻击和CNN网络训练的代码
Video Presentation文件夹中包含了攻击、防御、跳频攻击和CNN防御实现的视频
model_UI.py包含可执行文件界面UI的设计代码文件
